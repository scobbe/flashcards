## scan a code
### sǎomǎ
---
- **traditional:** 掃碼
- **definition:** scan a code
- **pinyin:** sǎomǎ
- **components:**
  - 扫(掃)
    - sǎo
    - sweep
  - 码(碼)
    - mǎ
    - code
- **etymology:**
  - **type:** compound word
  - **description:** 扫(掃) (sǎo, "sweep") + 码(碼) (mǎ, "code")
  - **interpretation:** Scan a code is derived from the action of sweeping (扫/掃) the surface to read a code (码/碼), typically with a digital device. This action mimics the sweeping motion in operating devices like scanners or smartphones to read QR codes or barcodes.
  - **simplification:** 扫(掃) for 掃 and 码(碼) for 碼
- **examples:**
  - 请(請)你扫码(掃碼)，获(獲)取优(優)惠信息。
    - Qǐng nǐ sǎomǎ, huòqǔ yōuhuì xìnxī.
    - Please scan the code to receive discount information.
  - 他在商店扫码(掃碼)，查看商品详(詳)情。
    - Tā zài shāngdiàn sǎomǎ, chá kàn shāngpǐn xiángqíng.
    - He scanned the code in the store to check the product details.

### 扫(掃)码(碼) → 码(碼)
- **traditional:** 碼
- **definition:** code
- **pinyin:** mǎ
- **components:**
  - 石
    - shí
    - stone
  - 马(馬)
    - mǎ
    - horse
- **etymology:**
  - **type:** phono-semantic compound
  - **description:**
    - Semantic: 石 (shí, 'stone') + Phonetic: 馬 (mǎ, 'horse') =
    - used for counting stones ->
    - number, code
  - **interpretation:** The character 码(碼) combines 石 (shí, 'stone') symbolizing things that can be counted, with 馬 (mǎ, 'horse'), chosen as a phonetic match. Together, they formed a character initially used for counting objects like stones, which later evolved to mean 'number' or 'code'.
  - **simplification:** The simplified character uses the exact same components as the traditional one.
- **examples:**
  - 请输(請輸)入密码(碼)。
    - Qǐng shūrù mìmǎ.
    - Please enter the password.
  - 系统(統)自动(動)生成验证码(驗證碼)。
    - Xìtǒng zìdòng shēngchéng yànzhèngmǎ.
    - The system automatically generates a verification code.
  - 扫(掃)描二维码(維碼)，即可获(獲)取信息。
    - Sǎomiáo èrwéi mǎ, jí kě huòqǔ xìnxī.
    - Scan the QR code to get the information.
