## scan a code
### sǎomǎ
---
- **definition:** scan a code
- **pinyin:** sǎomǎ
- **components:**
  - 扫(掃)
    - sǎo
    - sweep
  - 码(碼)
    - mǎ
    - code
- **etymology:**
  - **type:** compound word
  - **description:** 扫(掃) (sǎo, "sweep") + 码(碼) (mǎ, "code")
  - **interpretation:** Scan a code is derived from the action of sweeping (扫/掃) the surface to read a code (码/碼), typically with a digital device. This action mimics the sweeping motion in operating devices like scanners or smartphones to read QR codes or barcodes.
  - **simplification:** 扫(掃) for 掃 and 码(碼) for 碼
- **examples:**
  - 请你扫码(請你掃碼)，获取优惠信息(獲取優惠信息)。
    - Qǐng nǐ sǎomǎ, huòqǔ yōuhuì xìnxī.
    - Please scan the code to receive discount information.
  - 他在商店扫码(他在商店掃碼)，查看商品详情(查看商品詳情)。
    - Tā zài shāngdiàn sǎomǎ, chá kàn shāngpǐn xiángqíng.
    - He scanned the code in the store to check the product details.
