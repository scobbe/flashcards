## 扫码
---
- **pinyin:** sǎomǎ
- **definition:** scan a code
- **characters:**
  - 扫(掃)
    - sǎo
    - sweep
  - 码(碼)
    - mǎ
    - code
- **etymology:** Combines 扫(掃) (sǎo, "sweep") with 码(碼) (mǎ, "code") to express scan a code.
- **examples:**
  - 请扫码付款(請掃碼付款)。
    - qǐng sǎomǎ fùkuǎn
    - Please scan the code to pay.
