## first time
### dì yī cì
---
- **definition:** first time
- **pinyin:** dì yī cì
- **components:**
  - 第
    - dì
    - prefix for ordinal numbers
  - 一
    - yī
    - one
  - 次
    - cì
    - time; occurrence
- **etymology:**
  - **type:** compound word
  - **description:** 第 (dì, "prefix for ordinal numbers") is a prefix used to indicate the order or rank in a sequence, such as 'first', 'second', etc. 一 (yī, "one") specifies the number 'one'. 次 (cì, "time; occurrence") refers to an instance or occurrence of an event. Combined, 第 (dì, "prefix for ordinal numbers") + 一 (yī, "one") + 次 (cì, "time; occurrence") literally translates to the 'first instance' or 'first occurrence', thus meaning 'first time'.
  - **interpretation:** 第 (dì, "prefix for ordinal numbers") provides the ordinal sense, indicating position or sequence in a list. Following it, 一 (yī, "one") specifies the position as 'first', establishing the sequence. 次 (cì, "time; occurrence") then gives the notion of an event or instance. This leads to the understanding of 第一次 as 'the first event in a sequence' or simply 'the first time'.
- **examples:**
  - 这(這)是( )我( )第( )一( )次( )去( )北( )京( )。
    - Zhè shì wǒ dì yī cì qù Běijīng.
    - This is my first time going to Beijing.
  - 他第一次看到大海，感( )到( )非( )常( )兴(興)奋(奮)。
    - Tā dì yī cì kàn dào dàhǎi, gǎndào fēicháng xīngfèn.
    - He felt very excited when he saw the sea for the first time.
  - 我( )们(們)第( )一( )次( )见(見)面( )，是( )在( )一( )个(個)朋( )友( )的( )聚( )会(會)。
