## software
---
- **traditional:** 軟件
- **simplified:** 软件
- **pronunciation:** ruǎn jiàn
- **definition:** software
- **contemporary usage:**
  - 软件开发(软件開發)
    - ruǎnjiàn kāifā
    - software development
  - 软件工程
    - ruǎnjiàn gōngchéng
    - software engineering
  - 软件更新
    - ruǎnjiàn gēngxīn
    - software update
  - 软件测试
    - ruǎnjiàn cèshì
    - software testing
- **etymology (軟件):**
  - **type:** semantic compound
  - **description:**
    - semantic: 软(軟) (ruǎn, "soft") + 件 (jiàn, "item")
  - **interpretation:** A modern calque of English "software": 软(軟) (ruǎn, "soft") + 件 (jiàn, "item") mirrors "soft" + "ware" to denote programs. 软(軟) contrasts with 硬 (yìng, "hard") in computing, while 件 marks a discrete piece, yielding the sense of intangible program items.
  - **component characters:**
    - 软(軟)
      - ruǎn
      - soft
    - 软(軟)
      - ruǎn
      - soft
    - 件
      - jiàn
      - item; piece
  - **reference:** https://en.wiktionary.org/wiki/%E8%BB%9F%E4%BB%B6
  - **simplification rule (软件):** Simplified by replacing the 車 component in 軟 with 车 to form 软; 件 remains unchanged.
