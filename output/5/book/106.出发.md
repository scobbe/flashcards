## to set out; to start off
---
- **traditional:** 出發
- **simplified:** 出发
- **pronunciation:** chūfā
- **definition:** to set out; to start off
- **contemporary usage:**
  - 出发(發)时(時)间(間) (chūfā shíjiān, "departure time")
  - 出发(發)点(點) (chūfā diǎn, "starting point")
  - 出发(發)大(大)厅(廳) (chūfā dàtīng, "departure lounge")
  - 出发(發)地 (chūfā dì, "place of departure")
- **etymology (出發):**
  - **type:** semantic compound
  - **description:** semantic: 出 (chū, "go out") + 发(發) (fā, "depart").
  - **interpretation:** Combines 出 ‘go out’ with 发 ‘depart’ to express initiating movement from a place. Both morphemes contribute meaning, yielding the modern sense ‘to set out/start off’.
  - **component characters:**
    - 出
    - 发
    - 發
  - **reference:** https://en.wiktionary.org/wiki/%E5%87%BA%E7%99%BC
  - **simplification rule (出发):** The simplified form replaces 發 with the simpler 发 to reduce strokes, while 出 remains unchanged.
